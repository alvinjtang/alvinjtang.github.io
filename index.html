ABOUT ME

The quadratic formula. What is it? At this point, you either know or you don't care. But for me, it's the first function I ever programmed in my TI-83 in Mrs. Smith's 7th grade Algebra class. It's the first time I realized that I can talk to a machine and make it think for me. My curiosity was piqued and the quadratic function evolved to more complex projects like flipping a junk yard arcade cabinet into a multi-platform gaming machine. When college came, I decided to explore a different world of science; medicine, which eventually led me back to tech. I worked on a research team that was integrating neuroscience, biotech and gaming to innovate unconventional rehabilitation therapies for post-stroke motor recovery. After some years, I craved more tech forward work to improve/enhance the human experience with new technologies. So I decided to become a software engineer and attended Fullstack Academy of Code where I became equipped with the foundational knowledge to move forward with innovating new software and technologies as well as learning new languages and more complex frameworks.

PROJECTS

Chaindora

Chaindora is a music streaming service that benefits creators by storing track ownership and streaming revenue payout arrangements in a smart contract on the Etherium blockchain. Chaindora works as a conventional streaming service for listeners (who can pay by track or by subscription). For creators, it hosts tracks on IPFS, saves a file hash and creator id on the blockchain with a smart contract governing playback revenue payout.

  

Little Pizza Shop

Your one-stop shop for delicious authentic pizzas! Little Pizza Shop is a single-page web application project built with React for seamless instant rendering, Redux for state management, Node as the back-end runtime environment, Express for RESTful API routing, PostgreSQL relational database to store user and pizza data, and Sequelize ORM for data queries.

  

Index Owl

Making a career change? A little shaky about your technical vocab? Studying for your next technical interview? Just need a refresher? Index Owl is an open single-page web application built on the NERD stack. Any user is welcome to peruse the collections of flash cards to study, add to the collection themselves and delete any irrelevant or wrong definitions!

  

raft.AR

raft.AR is a web application that creates a virtual “business card” in augmented reality that renders user credentials in 3D floating in space and is easily accessible by anyone via a URL. raft.AR was built with Three.js for 3D modelling and rendered in augmented reality using 8th Wall Web's AR web platform.
