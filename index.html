PROJECTS

Map Big Deal Moments

Q2 2020 Priceline Hackathon as Best Overall Project

This was an effort between myself and fellow team engineer, Rachel Lee. For this 2-day hackathon Rachel Lee and I created a feature that allows customers to visualize their trip, with optimized driving routes and share a screenshot of the journey on social media or by email with friends and family.We leveraged the Google Maps Directions API, our internal typeahead, and design system to build a feature on top of the existing map view to create an innovative road trip planning experience for customers. We used Redux for state management to store an array of objects containing location information including latitude and longitude that allowed the Google Maps API to place markers and map out an efficient route for users to visualize their upcoming road trip. This project also generated interest amongst CEO, CTO and CPO to implement a robust version as a collaborative effort across multiple teams.

Index Owl

Making a career change? A little shaky about your technical vocab? Studying for your next technical interview? Just need a refresher? Index Owl is an open single-page web application built with React, Redux, Node, Express, Sequelize, and PostgreSQL. Pure CSS was used for all styling and card flipping animations. Any user is welcome to peruse the collections of flash cards to study, add to the collection themselves and edit any cards that need updating!

  

raft AR

RAFT.AR is a web application that creates a virtual “business card” in augmented reality that renders user credentials in 3D floating in space and is easily accessible by anyone via a URL. RAFT.AR was built with Three.js for 3D modelling and rendered using 8th Wall Web's AR web platform.

  

Little Pizza Shop

Your one-stop shop for delicious authentic pizzas! Little Pizza Shop is a single-page e-commerce web application built with React, Redux, Node, Express, Sequelize, PostgreSQL and Stripe for payment processing.
